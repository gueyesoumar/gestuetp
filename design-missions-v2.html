Recommandation

Combiner les 3 vues avec un switch :

  • Vue Kanban (A) — par défaut, pour voir tout le pipeline d'un coup et suivre la progression
  • Vue Split (B) — pour travailler sur une mission spécifique sans quitter la liste
  • Vue Cartes (C) — pour un aperçu riche avec les détails visuels

Le switch (3 boutons icônes) est en haut à droite, la préférence de l'utilisateur est sauvegardée.
